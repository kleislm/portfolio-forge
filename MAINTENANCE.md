# Maintenance / Manutenção

🇬🇧 This repository is published as **`kleislm/portfolio-forge`** on GitHub (renamed from `portfolio-pm`).

### Update content
```sh
cd ~/Downloads/portfolio-pm   # local folder kept; remote is portfolio-forge
# edit files…
git add -A && git commit -m "update" && git push
```
> Pushing requires a valid GitHub credential. Revoke old tokens after use.

### Make it public (only when you decide)
GitHub → repo → Settings → General → Danger Zone → Change visibility → Public.

### Optional: serve as a website
Enable **GitHub Pages** (or deploy the marketing site project) once public.
