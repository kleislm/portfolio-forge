# Artifacts

🇬🇧 English · [🇧🇷 Português](#-português)

Space for real artifacts that prove method (not just outcome). Suggested contents:

- **Priorization** — a real RICE/ICE export.
- **Opportunity Solution Tree** — a real tree for one product.
- **PR-FAQ** — a press release + FAQ (Working Backwards).
- **AI eval suite** — test cases + LLM-as-judge criteria (anti-hallucination, grounding).
- **North Star + input metrics** — the north-star definition and guardrails.

> In a product interview, one real artifact is worth more than five paragraphs. Artifacts here are sanitized — no confidential business data.
